1. 访问: https://github.com/new

2. 填写信息:

   - Description: 大宗交易信息检索平台

┌────────────────────────────────────────────────────────────────────────────┐
│ 步骤 2: 连接远程仓库并推送                                                  │
└────────────────────────────────────────────────────────────────────────────┘

GitHub 会显示命令，复制并在 PowerShell 中执行：

  git remote add origin https://github.com/你的用户名/仓库名.git
  git branch -M main
  git push -u origin main

或者直接执行（替换为你的仓库地址）：

  git remote add origin https://github.com/你的用户名/block-trade-platform.git
  git push -u origin main

┌────────────────────────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────────────────────────┘

2. 点击 "New Project"

3. 选择区域: Hong Kong (推荐)

4. 点击 "Add Service" → "Git"

5. 授权 GitHub 并选择你的仓库

6. 等待构建完成 (5-10分钟)
